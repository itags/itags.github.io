User guides

This project is under construction: al lot off code, examples, unit-tests and api need to be created and optimized. Yet the Itag-core is stable and operational, so we are currently working on a lot of Itags that will find their way in the final library.

Itags is a community-project. We develop basic commonjs Itag-modules which can be used in the browser. All modules should be also are serverside renderable. All modules are highly tested, well documented and highly performant. Among the modules are:

  • i-checkbox
  • i-select
  • i-scroller
  • i-table
  • i-chart-line

Functionality and state of HtmlElements is set at the attributes as much as possible. Read more here: State and rendering.

Quick Start

Itags provides a standard distribution which aggregates the various Itags. You can use itags as long as you include the loaderfile. Once included, you can also have a root namespace ITSA that has all functionality of the ITSA modules framework.

Customize

Instead of using the standard distribution, you can create customized build-files. This saves you from a large rollup with all available modules. Read more about it at Itags customized build (only for experienced users).
